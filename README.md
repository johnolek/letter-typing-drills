# Letter Drill

A typing drill app for practicing individual letters. Pick a letter set, drill by pressing the correct key, and track your speed per letter.

**[Try it live](https://johnolek.github.io/Letter-Drill/)**

## Development

```
npm install
npm run dev
npm run build
```

Builds to a single `docs/index.html` via `vite-plugin-singlefile`.

## Debug

Add `?startLevel=N` to the URL to start the streak meter at a specific level (e.g. `?startLevel=10`).
